# Inventory Management System

A simple web-based inventory management system built to help manage products, stock, and activity in a clean dashboard interface.

## Overview

This application allows users to manage product inventory through a modern dashboard.
Users can view product data, track stock, and manage inventory items easily.

The project focuses on simplicity, speed, and a clean user interface suitable for small inventory systems.

## Features

* Dashboard overview
* Product management
* Add new products
* Inventory activity tracking
* Clean and responsive admin interface

## Technologies Used

* **Next.js** – React framework for building the application
* **React** – Component-based UI library
* **Tailwind CSS** – Utility-first CSS framework
* **React Icons** – Icon library

## Installation

Clone the repository:

```bash
git clone https://github.com/HaekalAR/sistem-inventaris-s2.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
